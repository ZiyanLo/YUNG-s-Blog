# 个人博客网站 - 思想碎片

一个受 Yews.News 启发的极简主义个人博客网站，将艺术与思想完美结合。

## 🎨 设计理念

### Yews.News 风格特点
- **极简主义设计**：干净简洁的界面，专注内容展示
- **时间驱动内容**：通过时间段组织文章，创造阅读仪式感
- **艺术融合**：每篇文章都配有独特的艺术背景
- **移动端适配**：响应式设计，完美适配各种设备

### 网站特色
- **三个时间段**：清晨阅读(08:00)、午后思考(14:00)、深夜感悟(20:00)
- **五大分类**：科技、艺术、生活、哲学、全部
- **沉浸式阅读**：模态框设计，提供无干扰的阅读体验
- **艺术展示**：侧边栏艺术作品展示区

## 🚀 功能特性

### 核心功能
- ✅ 响应式设计，适配桌面端和移动端
- ✅ 分类筛选功能
- ✅ 文章详情模态框
- ✅ 平滑滚动和动画效果
- ✅ 时间指示器（根据实际时间变化）
- ✅ 特色文章推荐
- ✅ 艺术作品展示

### 交互特性
- 🎯 点击文章卡片查看详细内容
- 🎯 分类导航切换
- 🎯 头部导航栏智能隐藏/显示
- 🎯 ESC 键关闭模态框
- 🎯 悬停效果和过渡动画

## 🎯 设计原则

### 1. 极简主义 (KISS原则)
- 界面简洁，无多余装饰
- 专注内容，减少视觉干扰
- 直观的用户交互

### 2. 移动优先 (Mobile First)
- 响应式设计
- 触摸友好的交互
- 性能优化

### 3. 用户体验至上
- 平滑的动画过渡
- 直观的导航结构
- 沉浸式阅读体验

### 4. 艺术与技术的融合
- 色彩丰富的渐变背景
- 优雅的字体搭配
- 和谐的视觉层次

## 📱 响应式设计

### 断点设计
- **桌面端** (>1024px)：双栏布局
- **平板端** (768px-1024px)：单栏布局，侧边栏重新排列
- **移动端** (<768px)：完全单栏布局

### 适配特性
- 流式网格布局
- 弹性字体大小
- 触摸优化的交互元素

## 🎨 色彩方案

### 主色调
- **主色**：#1a1a1a (深灰)
- **辅色**：#666 (中灰)
- **强调色**：#333 (深灰)
- **背景色**：#ffffff (白色)
- **卡片背景**：#fafafa (浅灰)

### 功能色彩
- **成功状态**：#4CAF50 (绿色)
- **警告状态**：#FF9800 (橙色)
- **强调状态**：#9C27B0 (紫色)

## ⚡ 性能优化

- **CSS 优化**：使用 CSS 变量，减少重复代码
- **JavaScript 优化**：事件委托，减少内存占用
- **图片优化**：渐变背景代替图片，减少 HTTP 请求
- **动画优化**：使用 CSS transform 和 transition

## 🚀 使用方法

1. **直接打开**：双击 `index.html` 文件在浏览器中打开
2. **本地服务器**：使用 Live Server 等工具启动本地服务器
3. **部署上线**：上传到任何静态文件托管服务

### 本地开发
```bash
# 使用 Python 启动简单服务器
python -m http.server 8000

# 或使用 Node.js
npx serve .

# 访问 http://localhost:8000
```

## 📝 内容管理

### 添加新文章
在 `script.js` 文件中的 `blogData` 对象中添加新的文章：

```javascript
{
    id: 10,
    category: 'tech',
    title: '文章标题',
    summary: '文章摘要',
    content: '文章详细内容...',
    time: '08:00',
    date: '2024-01-15',
    artwork: 'linear-gradient(...)',
    source: '来源'
}
```

### 自定义分类
在 `categories` 对象中添加新的分类：

```javascript
const categories = {
    // 现有分类...
    newcategory: '新分类名称'
};
```
